## Best Practices

### 1. Choose the Right Type
- **Patent**: Only for actual patents (filed or granted)
- **Award**: For formal recognitions and competitions
- **Success Goal**: For measurable achievements (numbers, metrics)
- **Upgrade**: For version releases and major improvements
- **Update**: For everything else (partnerships, news, etc.)

### 2. Be Specific
- Use clear, descriptive titles
- Fill in all relevant optional fields
- Provide detailed descriptions
- Include context and impact

### 3. Attach Evidence
- Upload certificates for awards
- Include reports for goals
- Attach documentation for patents
- Add photos when relevant

### 4. Keep It Updated
- Add achievements as they happen
- Don't wait to batch them
- Update status for pending items
- Add follow-up achievements

## Quick Tips

### For Patents:
- Always include patent number if available
- Update status as it progresses (Filed â†’ Pending â†’ Approved â†’ Granted)
- Attach official documentation
- Explain the innovation and its impact

### For Awards:
- Include full award name and organization
- Specify the category or competition
- Attach certificate and photos
- Mention number of competitors if impressive

### For Success Goals:
- Use specific numbers (not "many" or "lots")
- Show progress (achieved vs target)
- Explain how you achieved it
- Include analytics or proof

### For Upgrades:
- Use semantic versioning (v1.0, v2.0)
- List key new features
- Explain improvements
- Attach release notes

### For Updates:
- Be newsworthy
- Include relevant details
- Attach press releases
- Keep it professional

## File Attachment Guidelines

### Supported Formats:
- **Images**: JPG, PNG, GIF, etc.
- **Documents**: PDF, DOC, DOCX, TXT
- **Maximum Size**: 10MB per file
- **Multiple Files**: Yes, unlimited

### Recommended Attachments:

**Patents**:
- Patent certificate (PDF)
- Technical drawings (PDF/Images)
- Filing documents (PDF)

**Awards**:
- Award certificate (PDF/Image)
- Event photos (Images)
- Press releases (PDF)
- Media coverage (PDF)

**Success Goals**:
- Analytics screenshots (Images)
- Reports (PDF)
- Dashboards (Images)
- Testimonials (PDF)

**Upgrades**:
- Release notes (PDF)
- Feature documentation (PDF)
- Demo videos (link in description)
- Screenshots (Images)

**Updates**:
- Press releases (PDF)
- Announcements (PDF)
- Agreements (PDF)
- Photos (Images)
